# Calculadora Dev Internacional

A calculadora definitiva para quem quer saber quanto vai receber dólares, euros ou Bitcoins em reais e vice-versa!

## Principais recursos

 * Conversor de moedas entre dólar, euro e BTC
 * Cálculo de salário anual para mensal e vice-versa.
 * Cálculo de salário mensal e anual baseado em horas diárias.
 * Entre outros!
